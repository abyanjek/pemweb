ABOUT ME

Perkenalkan, saya Abyan Zaky Danur Saputra, mahasiswa dari Program Studi Teknik Informatika di Universitas Indonusa Esa Unggul. Saat ini saya berada di semester kedua dan sedang mengikuti mata kuliah Pemrograman Web.

Saya memiliki latar belakang pendidikan di bidang Rekayasa Perangkat Lunak (RPL), yang memperkenalkan saya pada dasar-dasar pemrograman sejak SMK. Saya terbiasa menggunakan HTML, CSS, dan JavaScript secara dasar.

Melalui kuliah ini, saya ingin memperdalam pemahaman saya mengenai pengembangan aplikasi web, baik dari sisi tampilan (front-end) maupun fungsionalitas (back-end).
